# QuickAI 🚀

QuickAI is an AI-powered SaaS platform built using React, Vite, Node.js, Express, MongoDB, and Google Gemini AI.

## Features

* User Authentication
* AI Article Generator
* AI Resume Analyzer
* AI Text Summarizer
* AI Email Generator
* AI Image Generator
* Background Remover
* Object Removal Tool

## Tech Stack

* React.js
* Vite
* Node.js
* Express.js
* MongoDB Atlas
* Google Gemini AI
